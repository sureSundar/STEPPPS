# STEPPPS Schema

Canonical JSON schemas for TBOS STEPPPS contracts live here.

This location follows the decision recorded in `wisdom.md` Dialogue 12 and
contemplated by CX in Dialogue 13:

- `steppps-let.v1.schema.json` defines the atomic infolet/entity shape.
- `sangha-contract.v1.schema.json` defines local community governance.
- `nation-contract.v1.schema.json` defines larger governance boundaries.

Existing schemas under `steppps/schema` and `steppps/schemas` remain useful
historical and runtime artifacts. This directory is the stable include-level
contract surface for validators, generators, and kernel/userland gates.
