# Earth Entity Templates

Earth templates describe real-world beings, places, resources, events, and
authorities that can be represented as specialized STEPPPS-lets.

These templates are intentionally small. A full entity should still carry the
seven STEPPPS dimensions through `steppps-let.v1.schema.json`; the Earth
template gives the entity a more precise real-world kind.

Initial v1 templates:

- `human.v1.schema.json`
- `machine.v1.schema.json`
- `device.v1.schema.json`
- `place.v1.schema.json`
- `resource.v1.schema.json`
- `event.v1.schema.json`
- `authority.v1.schema.json`

Design rule: specialize without fragmenting the ecosystem.
